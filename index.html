Bienvenue sur le Memmory !

Venez jouer avec nous est faites le meilleur score possible 😎.

Règles du jeu :

  • Retourner 2 cartes 🃏. Si les images sont identiques, le jeu laisse les cartes visibles.
  • Si les images sont différentes ❌, les cartes ce retourne face cachées la ou elle étaient.
  • La partie est terminée lorque toutes le pairs ont été assemblées par pairs.

    • Si vous voulez vous souvenir de votre score, Inscrivez vous via le formulaire -> S'inscrire

    Le jeux garde les 5 meilleur score

    Si vous voulez relancer une partie cliquer sur la barre espace
